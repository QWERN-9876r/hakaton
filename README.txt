# Family Wallet

Web-приложение, позволяющее вести учёт семейных финансов, фиксировать поступление и использование средств.

Репозиторий на github: https://github.com/QWERN-9876r/hakaton
Тестовый стенд: http://liceum.ipmmp2019.ru (здесь можно оценить работоспособность приложения,
для входа можно использовать почту: testuser@test.ru и пароль 12345678, или же зарегистрировать свой аккаунт)

Предупреждение:
В тестовом стенде не работает авторизация через Google т.к компания требует https для общедоступных сайтов однако при запуске локально такого ограничения нет.


## О приложении
Хотите контролировать все расходы? Пользуетесь блокнотом для планирования  семейного бюджета? Храните данные о покупках в заметках телефона? Попробуйте Family Wallet - отличное решение контроля и планирования финансовых операций.

## Возможности
- Добавление источников дохода и категорий расходов.
- Детализация доходов и расходов.
- Статистика данных за месяц/год/всё время.
- Возможность объединения аккаунтов в группу (семью).
- Объединение доходов и расходов семьи.
- Поддержка валют: рубль, доллар, евро. Автоматическая конвертация.
- Регистрация в системе для синхронизации данных между устройствами.
- Создание общих источников дохода.
- Поддержка языков: английский, русский.
- Отображение данных в виде удобных графиков.

## Инструкция по развёртыванию проекта

### Развёртывание клиентской части

Для работы авторизации через Google и Яндекс необходимо ввести токены авторизации в файле .env.local в папке src.
При добавление авторизации через Google необходимо заполнить поля следующем способом:
Authorized JavaScript origins: http://localhost:3000
Authorized redirect URIs: http://localhost:3000/api/auth/callback/google
Yandex:
Redirect URI: http://localhost:3000/api/auth/callback/yandex

hakaton/client/hakaton/src/.env.local
```
GOOGLE_CLIENT_ID="<id Google аккаунта>"
GOOGLE_CLIENT_SECRET="<secret Google аккаунта>"
YANDEX_CLIENT_ID="<id Yandex аккаунта>"
YANDEX_CLIENT_SECRET="<secret Yandex аккаунта>"

NEXTAUTH_SECRET="supersecret"
NEXTAUTH_URL="http://localhost:3000"

SERVER_URL="http://localhost:3001"
```
Для работы Next js требуется [node js LTS версии](https://nodejs.org) и ввод в терминал команд приведенных ниже.
```
cd client/hakaton
npm ci
npm run build
npm start
```

### Развёртывание серверной части
Для корректной работы серверной части необходимо установить [Mongo DB](https://www.mongodb.com/docs/manual/installation/) последней версии
и выполнить следующие команды:
```
cd server
npm ci
npm start
```
